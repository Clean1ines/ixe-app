# План рефакторинга FileLinkProcessor

## Текущая сложность: C (16)

## Компоненты для выделения:

### 1. FileLinkExtractor
- Извлечение ссылок на файлы из HTML
- Фильтрация по расширениям (.pdf, .doc, .zip и т.д.)
- Возврат списка кандидатов для загрузки

### 2. FileDownloadCoordinator  
- Управление параллельными загрузками
- Семафор для ограничения concurrent downloads
- Координация задач загрузки

### 3. FileDownloader
- Загрузка отдельных файлов
- Обработка ошибок загрузки
- Сохранение файлов в указанную директорию

### 4. HtmlLinkUpdater
- Обновление ссылок в HTML после загрузки
- Замена абсолютных URL на относительные пути

### 5. FileLinkProcessor (фасад)
- Координация всех компонентов
- Сохранение обратной совместимости

## Ожидаемый результат:
- Снижение сложности с C (16) до A (<10)
- Улучшение тестируемости каждого компонента
- Сохранение существующего API
